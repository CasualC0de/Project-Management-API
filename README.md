# Project Management API

ASP.NET Core Web API for managing projects and tasks

## Tech Stack:
- ASP.NET Core
- PostgreSQL
- Entity Framework Core
- Docker

## Status
In Development.

## Roadmap
The current goal of this project, besides the purpose of learning, is to have a free and minimalistic developer-first PM system.
The targets are as follows:
- Clean UI
- Minimal configuration
- Simple workflows
- blocker detection
- auto-status transitions
- dependency graph between tasks
- task order suggestions(suggestions would be based on information such as "Due dates", "Priority", "Workload" and "Dependencies")
